# scoop-cvp

Custom scoop packages

## Usage

[Scoop](http://scoop.sh/) is apt-get (or homebrew) for Windows.

To add this bucket to scoop, type:

```
scoop bucket add cvp https://github.com/SayCV/scoop-cvp.git
```
